# Contributing to PathFinder

Thank you for your interest in PathFinder.

## Contribution Policy

Please note that PathFinder is currently an active academic/personal submission for the Hack Club Stasis hardware program. At this stage:

1. **Sole Contributor**: The project is solely designed, built, and maintained by **lakkeychoudhary**.
2. **Pull Requests**: Pull requests from outside contributors will not be merged at this time to maintain the academic integrity of the hackathon submission.
3. **Issue Reporting**: If you find bugs or have feature suggestions, please feel free to open an Issue in the GitHub repository.

## Development Standards

For any local development, please adhere to:
- Formatted Arduino C++ / ESP-IDF C++ code.
- Detailed schematic updates in target schematics directories.
- Clean updates to the Bill of Materials (BOM) CSV file in the root directory.

Thank you for your understanding.
